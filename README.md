## Hangul iPuTTY Project

한글 iPuTTY 프로젝트 홈페이지에 오신 것을 환영합니다.

이 프로젝트는 국제화된 PuTTY를 개발하기 위해 시작되었으며, 현재는 한국어 사용자들을 위해 특화된 기능들을 주로 제공하고 있습니다.
